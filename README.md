# 🎨 Фото + Видео бот

**Цены:** Фото 0.30₽ | Видео 0.30₽

## Команды

```
/photo кот          → фото
/video закат        → видео (наезд камеры, 5 сек)
/stats              → статистика
```

## Запуск

```bash
pip install -r requirements.txt
python bot.py
```
Не забудь вставить токены в .env

## Технологии

- SDXL Lightning (gen-api.ru)
- MoviePy
- Aiogram 3
